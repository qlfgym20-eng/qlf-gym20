# Dinateck Gym - Gestion de Salle de Sport

Application SaaS de gestion de salle de fitness (PWA).
- **URL** : https://sport.dolisoft.com
- **Stack** : React + Vite + Supabase + Workbox
- **Langues** : Français, Arabe

## Structure

```
├── index.html              # App shell (SPA)
├── manifest.webmanifest    # Web Manifest PWA
├── sw.js                   # Service Worker (Workbox)
├── push-handler.js         # Notifications push
├── registerSW.js           # Enregistrement SW
├── favicon.ico
├── pwa-192x192.png
├── pwa-512x512.png
├── placeholder.svg
├── pos/                    # Images produits POS (15)
├── assets/                 # Bundles JS/CSS (115+)
└── README.md
```
